## Field-by-field guide for new entries

- **Retailer**: human-readable name (e.g., "Alo Yoga")
- **Slug**: filename stem, lowercase + hyphens (e.g., `alo-yoga`). Must match `data/<slug>.json`.
- **Source URL**: the *exact* URL that worked, including any query params or path tricks (e.g., `/stores/all-X` directory page).
- **Platform**: `yext`, `brandify`, `storerocket`, `algolia`, `shopify`, `custom-nextjs-ssr`, `custom-api`, `static-html`, etc.
- **Scope**: `global`, `US+CA`, `EU only`, etc. — what slice of the world this scrape covers.
- **Stores**: total count, optionally with country breakdown.
- **Script**: path under `_scripts/`, e.g., `_scripts/scrape_alo-yoga.py`.
- **Gotchas**: anything that tripped us up. Bot protection? Regional split? Required headers? Weird address format? Missing fields? Pagination quirks? Rate limits? Unusual data paths inside `__NEXT_DATA__`?
